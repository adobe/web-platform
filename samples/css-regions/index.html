CSS Regions

About

The CSS Regions module allows content to flow across multiple areas called regions. The CSS regions module provides an advanced content flow mechanism, which can be combined with positioning schemes as defined by other CSS modules.

Why use CSS Regions

Web Designers and Developers

  • Take advantage of new CSS properties that let you build rich interfaces with multiple text flows.
  • CSS Regions are easy to pick up. They build upon familiar CSS properties and structure.

Print Designers

  • Bring high-fidelity print layouts to mobile devices and target multiple screen sizes with one highly adaptive layout.

Enable CSS Regions in Google Chrome

  1. Using the browser bar navigate to chrome://flags
    • If you're running Chrome 21 or Chrome 22, find the "Enable CSS Regions" flag and enable it
    • If you're running Chrome 23 or newer, find the "Enable experimental WebKit features" flag and enable it.
  2. Restart your browser by clicking the "Relaunch Now" button on the lower left corner.

Browser support

CSS Regions is currently under development and may be expected to change.

The most up-to-date CSS Regions feature set is implemented in the latest Webkit nightly build. Most features are also available in the Google Chrome browser after enabling the necessary flag.

The browser support matrix shows the current level of support of CSS Regions features across tested browsers.
